# SnakeGame

Консольная игра «Змейка», написанная на C# для .NET 8.

## Запуск

```powershell
dotnet run
```

## Управление

- Стрелки или WASD - движение змейки.
- P - пауза.
- R - начать заново после проигрыша.
- Esc - выход из игры.

## Правила

Игрок управляет змейкой, собирает еду `*` и получает очки. При столкновении со стеной или собственным телом игра завершается.
